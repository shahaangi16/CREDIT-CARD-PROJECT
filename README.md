# CREDIT-CARD-PROJECT

Credit Card Customer & Transaction Analytics (Power BI | SQL | Data Modeling)

Developed an enterprise-grade Power BI dashboard analyzing 55.4M+ annual revenue, 577M customer income, and 7.9M interest, leveraging two datasets (customer + transaction).

Conducted customer segmentation across 10K+ profiles, revealing high-value cohorts — Age 30–40 (14M), High salary group (15M), and Married customers contributing 14M revenue.

Mapped spending patterns: Bills (14M), Entertainment (10M), Fuel/Grocery (9M each) as dominant expenditure categories.

Evaluated card performance: Blue card generated 4.62M revenue, with 78.5K total interest earned and 29.5K annual fees collected.

Built quarterly trend visuals showing Q3 peak revenue (14.2M) and Q4 peak transaction volume (166.6K).

Created job-wise profitability insights: Self-employed (14M), Businessman (11M), Blue-collar (9M), Govt (9M).

Delivered geographic insights ranking top revenue states: TX (10M), CA (7M), NY (6M).

Improved executive decision-making by designing 20+ interactive visuals filtered by card category, quarter, salary class, and expenditure type.
